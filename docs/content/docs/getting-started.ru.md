Зарегистрируйтесь на JetSwap и начните зарабатывать кредиты для рекламы своего сайта. Полная инструкция для новичков.

## Регистрация на платформе

1. **Перейдите по кнопке "Регистрация"** на главной странице
2. **Введите email** — на него придёт код подтверждения
3. **Введите полученный код** и нажмите "Продолжить регистрацию"
4. **Заполните профиль:**
   - Логин
   - Пароль (10–50 символов)
   - Адрес сайта (можно изменить позже)
   - WebMoney кошелёк (опционально)

**Получите сразу 100 стартовых кредитов!**

## Первые шаги после регистрации

{{% steps %}}
### 1. Войдите в дашборд
После регистрации автоматически откроется ваш личный кабинет со статистикой и балансом.

### 2. Настройте браузер
- Используйте **Google Chrome** последней версии
- Включите JavaScript, графику, Cookies
- Отключите блокировщики рекламы (AdBlock)

### 3. Начните серфинг
```
Перейдите в раздел "Серфинг" → 
Нажмите "Начать ручной серфинг" → 
Получайте 2–4 кредита за просмотр!
```

### 4. Добавьте свой сайт
```
Дашборд → "Мои сайты" → 
Введите URL → Отправьте на модерацию (1–24ч)
```

### 5. Запустите первую рекламу
```
Баланс > 100 кредитов → 
"Мои кампании" → Выберите бюджет → Старт!
```
{{% /steps %}}

## Настройка серфинга (важно!)

Перед началом работы убедитесь:

```yaml
✅ Один аккаунт = один IP
✅ Только одно окно серфинга
✅ Chrome без расширений
✅ Интернет > 10 Мбит/с
✅ JS + Cookies включены
```

## Частые ошибки новичков

| Проблема | Решение |
|----------|---------|
| Сайты не загружаются | Отключите AdBlock, включите графику |
| "Код неверный" | Внимательно смотрите цифру на картинке |
| "Много окон" | Закройте все вкладки серфинга |
| Просмотры не засчитываются | Ждите ровно 30 секунд |

## Разрешение проблем с доступом

**Если сайты для вашего IP закончились:**
1. Подождите 1–2 часа
2. Смените IP (VPN/мобильный интернет)
3. Переключитесь на задания
